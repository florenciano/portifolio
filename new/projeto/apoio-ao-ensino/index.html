Nesta página estão listados alguns projetos realizados para auxiliar a capacitação dos professores do Insper – contribuindo de forma direta na melhoria do ensino, quanto para comunicar ações ou iniciativas desenvolvidas na Instituição cujos professores foram o público-alvo.

Manuais de utilização

Um ambicioso projeto de transpor dezenas de documentações/tutoriais, em formato .pdf, para documentos web interativos. Estes documentos são usos de ferramentas tecnológicas educacionais: Blackboard Learn; Catme; Clickers (votadores eletrônicos); Casos, simuladores e Cursos online.

Desafios:

  • Proporcionar ao usuário uma forma de consulta rápida por meio de palavras-chaves (mindset dos usuários);
  • Oferecer uma versão adaptativa para dispositivos móveis;
  • Criar uma documentação de sistema para auxiliar na produção/organização e na manutenção futura destes materais;

Tecnologia front-end: AngularJS e Gridiculous Boilerplate.

Fev-Jun 2014

Interação 121 (one-to-one)

O projeto é a documentação de um relatório desenvolvido pela área responsável pelo Desenvolvimento de Ensino e Aprendizagem (DEA) do Insper, que conduziu ao longo do ano de 2016, conversas individuais com os professores. Foi criado Personas para representar grupos de professores que compartilham da mesma visão educacional. Os personas permitirão intervenções estratégicas institucionais.

Desafios:

  • Criar uma identidade visual e peças de comunicação;
  • Criar um material responsivo;
  • Elaborar template de Persona com representações de dados quanti e quali;
