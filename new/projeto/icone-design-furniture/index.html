Um ícone do Furniture Design

Design de Interface

 

Este é um projeto conceitual realizado para o escritório Eames Office LLC com o objetivo de incrementar o volume de vendas para um dos seus mais famosos projetos: Eames Chair Lounge – um ícone do furniture Design. É provavelmente a poltrona mais famosa de todos os tempos, foi projetada pelo casal Charles e Ray Eames na década de 50, e ocupa até hoje o seu espaço na coleção permanente do Moma, em Nova York.

A proposta foi criar uma landing-page que destaca seus principais atributos: modernidade e conforto. Também foi proposto pequenas interações de interface para que aos usuários explorarem ainda mais detalhes do produto.


Ago 2017

A paleta de cores foi um esforço em somar à modernidade um toque de sofisticação. Para evitar profusões limitei a quatro tons. Tonalidades quentes = moderno e contemporâneo. Tonalidades frias = sofisticação e luxo. As cores se intercalam para proporcionar equilíbrio.

A tipografia sans-serif é a categoria de fontes mais apropriada para tornar uma interface contemporânea. Escolhi a fonte Raleway (Google Fonts®), que apresenta uma ótima legibilidade. Seus caracteres numéricos é do tipo old style uma herança das tipografias tradicionais serifadas, isto cria um vínculo entre moderno e clássico.
